## 📖 Описание проекта
Целью работы является изучение структуры данных объявлений Airbnb, выявление ключевых закономерностей ценообразования и построение прогнозных моделей для предсказания стоимости аренды. 

**Ключевые задачи:**
* Провести разведочный анализ данных (EDA).
* Выявить географические и качественные факторы, влияющие на цену.
* Сравнить эффективность различных алгоритмов машинного обучения.

---

## 📊 Используемые данные
В исследовании используется датасет Airbnb New York City, доступный на платформе **Kaggle**.
* **Объем данных:** 48,895 записей.
* **Ключевые признаки:** район (`neighbourhood_group`), тип жилья (`room_type`), цена (`price`), координаты, количество отзывов и доступность.

---

## 🔍 Основные этапы исследования
1. **Первоначальный анализ:** Очистка данных, обработка пропусков в столбцах `name`, `host_name`, `last_review` и `reviews_per_month`.
2. **EDA:** Визуализация распределения цен. Установлено, что более 80% предложений находятся в диапазоне **$50–$200**.
3. **Корреляционный анализ:** Сравнение методов Пирсона и Спирмена. Выявлено, что тип жилья имеет наиболее сильную связь с ценой (корреляция Спирмена = 0.68).
4. **Геоанализ:** Создание интерактивных карт (Plotly) для визуализации ценовых кластеров. Самая высокая концентрация дорогих объектов зафиксирована в Манхэттене.

---

## 💡 Выдвинутые гипотезы
* **Географическая:** Цены в Манхэттене выше на 30–50% по сравнению с другими районами.
* **Тип жилья:** Весь дом/апартаменты стоят в 2–3 раза дороже, чем отдельная комната.
* **Рейтинговая:** Популярные объекты (с большим числом отзывов) имеют тенденцию к более низкой цене.

---

## 🤖 Результаты моделирования
Для предсказания цен были протестированы четыре модели градиентного бустинга и ансамблей. В качестве метрики использовалась **RMSE**.

| Модель | RMSE |
| :--- | :--- |
| **CatBoost Regressor** | **198.96** (Лучшая модель) |
| LightGBM | 202.27 |
| XGBoost | 207.93 |
| Random Forest | 211.39 |

После оптимизации гиперпараметров (`learning_rate`: 0.1, `iterations`: 800, `depth`: 4 и др.) **CatBoost Regressor** показал наилучший результат и высокую устойчивость к категориальным признакам.
